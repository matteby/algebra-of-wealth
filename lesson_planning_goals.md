# Project Brief: "The Algebra of Wealth" Interactive Seminar

## 1. Core Objective
The goal is to create a high-impact, interactive educational suite for a 1.5-hour seminar based on Scott Galloway’s book, *The Algebra of Wealth*. The focus is specifically on the sections: **Diversification: Basic Principles of Investing** and **Diversification: The Capitalist’s Handbook**.

## 2. Target Audience
* **Participants:** 4 advanced 8th-grade students.
* **Academic Level:** High-achievers operating at a 10th-grade academic level.
* **Context:** The content must be rigorous and data-driven, avoiding "junior" simplifications.

## 3. Required Artifacts
The project must produce the following three components:

### A. Teacher Notes
* A structured guide for the instructor to lead the 1.5-hour session.
* Include a precise timeline, high-level Socratic prompts, and technical "deep-dive" context for the math involved.

### B. Student Handouts
* Lightweight, high-retention summary sheets.
* Focus on the "Mental Models" of wealth preservation and core formulas.

### C. Interactive HTML Dashboard
* A web-based presentation tool to be used during the seminar.
* **Features:** * Interactive charts (e.g., compounding interest vs. fee impact).
    * "Mathematics of Loss" calculators ($G = \frac{1}{1 - L} - 1$).
    * Visualizations for diversification vs. concentration risk.
* **Tech Stack:** Vanilla HTML/CSS/JS (Tailwind CSS for styling is preferred).

## 4. Guiding Principles
* **Accuracy over Entertainment:** Use Galloway's specific logic (e.g., the asymmetry of loss).
* **Engagement through Simulation:** Shift from lecture to "hands-on" modeling via the HTML dashboard.
* **Sophistication:** Professional, "FinTech" aesthetic that respects the students' advanced cognitive abilities.
